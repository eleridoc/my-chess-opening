Export tools for the current Explorer position. This dialog shell is introduced in V1.10.2. The real data wiring and actions will be added in the next V1.10 tasks.

FEN

Exact current board position.

The copy action will be enabled in V1.10.4.

PGN

Current line up to the active cursor.

The PGN builder and copy action will be enabled in V1.10.3 and V1.10.5.

PNG

Export the current board rendering as an image.

V1.10.6 will first try to export the current `cm-chessboard` rendering as displayed (option A). If that is not technically reliable, the implementation will fall back to a simpler board image export.
