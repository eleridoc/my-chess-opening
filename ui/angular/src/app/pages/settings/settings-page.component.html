How to create the token

My Chess Opening only uses this token to authenticate requests to the Lichess Opening Explorer API.

  1. Open the Lichess personal access token creation page.
  2. Create a token name such as My Chess Opening.
  3. Do not check any permission box. The Opening Book does not need account actions.
  4. Copy the generated token and paste it below.

Leaving all permissions unchecked limits the risk if the token is leaked: it should not authorize account actions such as playing games, managing challenges, messages, preferences, or studies.
